# EcoShare
EcoShare is a community-based web application that allows people to lend and borrow items instead of buying new ones. The aim is to reduce waste, support the cost-of-living crisis, and encourage community sharing.

## Project Theme
Sharing, exchange, and building community

- Backend: Node.js, Express.js
- Database: MySQL
- DevOps: Docker, GitHub Actions
